# 🚀 Site Alienet

Projeto desenvolvido como parte do curso do SENAC.

## 📌 Sobre o projeto

Este projeto consiste em um site desenvolvido com HTML, CSS e JavaScript, com foco em layout moderno e navegação intuitiva.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do projeto

- index.html
- login.html
- usuario.js

## ▶️ Como executar

1. Baixe o projeto
2. Abra o arquivo `index.html` no navegador
